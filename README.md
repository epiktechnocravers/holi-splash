# 🎨 Holi Color Splash 2026

Interactive Holi greeting card generator. Throw virtual colors, create personalized cards, share on WhatsApp.

## Setup
Pure static site — just serve the files:
```bash
npx serve .
# or
python3 -m http.server 8080
```

## Features
- Touch/mouse color splashing with realistic blob shapes
- 8 vibrant Holi colors + random mode
- Personalized greeting card generation
- Download as PNG / Share on WhatsApp
- Mobile-first, works on all devices

## Deploy
Upload all files to any static host (Netlify, Vercel, GitHub Pages, etc.)
